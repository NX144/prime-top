Как найти фильм?

Из-за большого количества стриминговых сервисов пользователю часто приходится тратить время на поиски фильмов достаточно большое количество времени. При этом найдя фильм на каком либо из сервисов, может оказаться так, что у него нет подписки и нужно регистрировать аккаунт/выбирать тариф подписки /либо покупать фильм.

Сервис для поиска

Создать сервис(мобильное приложение + сайт) для одномоментного поиска необходимого видеоконтента с использованием текстового и графического(по фото) поиска. Который позволит получить исчерпывающую информацию о том на каком из стриминговых сервисов можно посмотреть искомый фильм, узнать стоимость фильма/подписки и.т.д  с возможностью прямого перехода на страницу необходимого контента.

Смотри, а не ищи

Разработка сайта и мобильного приложения позволяющего осуществлять поиск необходимого контента по всем популярным стриминговым сервисам.
